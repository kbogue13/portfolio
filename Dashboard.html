Operations Dashboard

Dashboards provide real-time, controlled, and accurate monitoring of key performance indicators and can offer significant benefits including better decisionmaking and a targeted allocation of resources.

To the create the dashboard below I constructed a database and empty feature class with my desired schema on my desktop computer. Then I pushed them to the cloud so that they could be accessed both on my mobile phone for data collection and by this dashboard to display the collected data in real time. I chose to display potholes but nearly any spatial data could be leveraged to create a similar application.
