Simple Google Map


Albania is an underrated Balkan country rich with ancient and natural beauty. I was fortunate enough to spend three weeks in Albania in July 2016 and of all the positive things I discovered about this little-known country, what stood out most was the friendliness of its citizens.

Using Google Maps Application Programming Interface (API), I was able to embed a web map into my site with a marker located at a point that interests me. The reason I am able to embed a non-static map in my page is becuase the API acts as an intermediary that allows different applications to "talk" to one another. In this case the API is allowing my page to "talk" to Google Maps and tell it to show a map centered on the point that I have chosen (Tirana, Albania).
